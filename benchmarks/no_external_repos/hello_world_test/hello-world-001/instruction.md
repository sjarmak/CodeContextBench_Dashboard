# Task: Create Hello World Script

Create a Python script named `hello_world.py` in the workspace root that prints "Hello, World!" to stdout.

## Requirements

1. Create file `hello_world.py`
2. The script should print exactly: `Hello, World!` (with exclamation mark)
3. Should work when run with: `python hello_world.py`

## Success Criteria

The test will verify:
- File exists at `/workspace/hello_world.py`
- Running the script outputs exactly "Hello, World!" followed by a newline
